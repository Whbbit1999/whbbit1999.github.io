对象

  • 对象：一切具有本质特征和行为的事物。

    • 属性（变量）：本质特征
    • 方法（函数）：行为

  • 构造函数（类）：具有相同特征和行为的一类事物。

类是对象的抽象化；对象是类的具象化

如何创建对象？

  1. 字面量创建
    {key: value, key : value}

  2. new Object()

  3. new 自定义构造函数()

    • 构造函数的格式
    function 大驼峰命名([参数]){
    this.属性: value,
    this.方法 = function(){
        ...
    }
}

工厂模式： - 缺点： 1. 无法分辨对象具体属于哪一类 2. 不符合程序创建对象的习惯

  2. 对象[‘属性’] 对象方法

数组

shift 方法删除所有元素
？如果数组为空，使用后的结果

概述

  1. 概念：用于存储一组或一系列相关数据的容器
  2. 下标：区别数组空间中唯一的索引值
  3. 数组元素：存储在数组中的数据段
  4. 作用：存储或处理大量数据‘
  5. 好处：集中管理，提高性能（寻址速度）

如何声明数组？

  1. 字面量： []
  2. new 构造函数
    • 如果构造函数有且仅有一个正整数，则表示数组的长度
    • 如果构造函数中有一个小数或负数，则报错
    • 如果构造函数中有一个非数字类型的数据，或多个数据，则表示数组元素

数组的属性

  • length 表示数组的长度

如何给数组赋值？

  1. 初始化数组：声明时同时赋值。
    • var arr = [1,2,4];
  2. 先声明，后赋值

遍历（循环）

方法：增删改截拼复排转 （作用、参数、返回值、是否影响原数组）

  • 增（添加元素）

    1. 前增

      • unshift(元素,元素,……)
      • 作用：在数组的首部增加元素
      • 返回值： 新增后数组的长度
      • 是否影响元素组：是
      var arr = [1, 2, 3, 4]
console.log(arr.unshift(-1, true)) //6
console.log(arr) //[-1,true,1,2,3,4]

    2. 后增

      • push(元素,元素,……)
      • 作用：在数组的尾部增加元素
      • 返回值：新增后数组的长度
      • 是否影响元素组：是
      var arr = [1, 2, 3, 4]
console.log(arr.push(7, 9)) // 6
console.log(arr) // [1,2,3,4,7,9]

  • 删（删除元素）

    1. 前删

      • shift()
      • 作用：删除数组的第一个元素
      • 返回值：删除的元素
      • 是否影响原数组： 是
      var arr = [1, 2, 3, 4]
console.log(arr.shift()) // 1
conaole.log(arr) // [2,3,4]

    2. 后删

      • pop()
      • 作用：删除数组的最后一个元素
      • 返回值：删除的元素
      • 是否影响原数组：是
      var arr = [1, 2, 3, 4]
console.log(arr.pop()) // 4
console.log(arr) // [1,2,3]

  • 改（）

    • splice(下标开始位置,删除元素的长度,元素,元素,……)
    • 作用：在数组的任意位置增、删、改
    • 返回值：被删除的元素
    • 是否影响元素组：是
    //一个元素
var arr = [1, 2, 3, 4]
console.log(arr.splice(1)) //[2,3,4]
console.log(arr) //[5]

//两个元素
var arr = [1, 2, 3, 4]
console.log(arr.splice(1, 2)) //[2,3]
concole.log(arr) // [1,4]

//多个元素（删除并在原位置插入新元素）
var arr = [1, 2, 3, 4]
console.log(arr.splice(1, 2, true, false, [1, 2, 3])) //[2,3]
console.log(arr) // [1,true,false,[1,2,3],4]

//多个元素（不删除，在指定位置插入新元素）
var arr = [1, 2, 3]
console.log(arr.splice(2, 0, true, false)) // []
console.log(arr) // [1,2,true,false,3,4]

  • 截（）

    • slice(下标开始的位置, 下标结束的位置)
    • 作用：截取指定范围的元素
    • 返回值：截取到的元素数组
    • 是否影响元素组：否
    //一个参数
var arr = [1, 2, 3, 4]
console.log(arr.slice(1)) // [2,3,4]
console.log(arr) // [1,2,3,4]

//两个参数（第一个小于于第二个）
var arr = [1, 2, 3, 4]
console.log(arr.slice(1, 3)) // [2,3]
console.log(arr) // [1,2,3,4]

// 两个参数（第一个大于第二个）
var arr = [1, 2, 3, 4]
console.log(arr.slice(3, 1)) // []
console.log(arr) // [1,2,3,4]

// 两个参数（负数） 表示倒数
var arr = [1, 2, 3, 4]
console.log(arr.slice(-3, -1)) //[2,3]
console.log(arr) //[1,2,3,4]

  • 拼

    • concat (元素,元素,……)

    • 作用：在数组后面拼接新元素

      如果新元素是一个数组，则展开最外层的数组拼接到后面

    • 返回值：拼接后的新数组

    • 是否影响原数组：否

    //新元素
var arr = [1, 2, 3, 4]
console.log(arr.concat(8, 9)) // [1,2,3,4,8,9]
console.log(arr) // [1,2,3,4]
//新元素是数组
var arr = [1, 2, 3, 4]
console.log(arr.concat(8, [1, 2, [5, 4], 7], 9)) // [1,2,3,4,1,2,[5,4],7,9]
console.log(arr) // [1,2,3,4]

  • 复（复制）

    // 截取数组复制
function fnCopyArr(arr) {
  return arr.slice(0)
}
//拼接空字符串复制
function fnCopyArr1(arr) {
  return arr.concat()
}
// 遍历数组复制
function fnCopyArr2(arr) {
  var list = []
  for (var i = 0; i < arr.length; i++) {
    list[i] = arr[i]
    // list.push(arr[i])
  }
  return list
}

  • 排（排序）

    1. reverse()
    • 作用：将数组中的元素逆序存放
    • 返回值：返回逆序后的数据
    • 是否影响原数组：是
    var arr = [100,23,4,6,2]
console.log(arr.reverse()); //[2,6,4,23,100]
console.log(arr); // [2,6,4,23,100]
    1. sort()

    • 作用：将数组按字符编码进行从大到小排序

    • 返回值：返回排序后的数组

    • 是否影响原数组：是

      • 使用 sort 将数组中的元素从小到大排列
      var arr = [100,23,4,6,2]
sort(function(a,b){return a-b})
console.log(arr) // [2,4,6,23,100]
      • 是用 sort 将数组中的元素从大到小排列
      var arr = [100,23,4,6,2]
sort(function(a,b){return b-a})
console.log(arr) // [100,23,6,4,2]
    var arr = [100,23,4,6,2]
console.log(arr.sort()) // [100,2,23,4,6]
console.log(arr) // [100,2,23,4,6]

  • 转（转成字符串）

    • 作用：将数组转换成字符串
    • 返回值：转换成的字符串
    • 是否影响原数组：否
    1. toString()

    toString 不是数组的方法，是 Object 对象的方法，数组是继承于 object

    • 将一个整数转化为制定进制的字符串
    var num = 16;
console.log(num.toString(2))// 2
console.log(num.toString(8))// 20
console.log(num.toString(16))// 10
console.log(num.toString(37))//报错，只能是在2~36之间
    • 将数组转为字符串
    var arr = [1,2,3,4]
console.log(arr.toString())// '1,2,3,4'
console.log(arr)// [1,2,3,4]

    1. join(‘连接符’)

堆栈（值传递和地址传递/引用传递）

堆和栈

堆：先进先出
栈：先进后出

简单辨别是值传递还是地址传递

  1. js 中的基本数据类型属于值传递（传递的是值）
  2. js 中的符合类型属于引用传递（他们之间传递的是引用堆的地址）

json

  • json 是跨语言、跨平台的一种网络数据传输格式
  1. []
    • 原生：[‘a’,’b’]
    • json：[“a”,”b”]
  2. {}
    • 原生：{name=”张三”}
    • json：{“name” = “张三”}

json 的 value 值不能是 NaN、undefined、function
在一个 json 文件中，只能写一个 json 数据
json 键值对必须用双引号

排序算法

  1. 冒泡排序：相邻的两个元数进行比较，符合条件，交换位置
  2. 选择排序：依次取出元素与剩下的所有元素进行比较，符合条件，交换位置
